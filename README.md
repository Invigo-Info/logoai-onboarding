# Logo.ai — Prelaunch Page (Next.js)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Editing Content

All copy is in **`app/page.tsx`** at the top of the file in clearly labeled content objects (`HERO`, `WHAT_YOU_GET`, `HOW_IT_WORKS`, etc.). Edit text there — the JSX below renders it automatically.

## Connecting Email Signup

`SignupForm` now posts to `/api/waitlist`, which uses the Supabase service role key on the server to insert rows safely (no RLS policy required). Populate the environment variables below in `.env.local`, then restart `npm run dev`:

```
NEXT_PUBLIC_SUPABASE_URL=your-project-url
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

Create a `waitlist` table inside Supabase (SQL snippet; `source` column is optional if you want to track signup origin):

```sql
create table public.waitlist (
  id bigint generated by default as identity primary key,
  email text unique not null,
  source text default 'landing-page',
  created_at timestamptz default timezone('utc', now())
);
```

If you already created the table without `source`, run:

```sql
alter table public.waitlist
add column if not exists source text default 'landing-page';
```

The service role key must stay server-only (never prefix it with `NEXT_PUBLIC`). If you need a different provider, replace the fetch call inside `SignupForm` with your own API request.

## Brand Kit Image

## Fonts

- **DM Sans** — body text (loaded via `next/font/google`)
- **Space Mono** — labels, tags, countdown
- **Satisfy, Pacifico** — decorative logo gallery fonts (loaded via CSS `@import`)
- Other display fonts — Oswald, Comfortaa, Fredoka, Bebas Neue, Playfair Display
